# LiteUI
Light, multi-style UI library allowing for easy UI writing.

LiteUI allows you to write UI in Fluent Imperative (chaining methods of instructions) or Declarative. (describing a UI and how it behaves)

Its power comes from its speed and weight, coming in batteries not included at default with a single module at a few hundred lines. To get said extras, you'd have to download the .rbxm containing the plugins.
